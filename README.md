# Brewfile

Auto backup Brewfile by launchd.

## Installation

```sh
$ curl -sL https://raw.githubusercontent.com/elzup/Brewfile/master/install.sh | sh
```

## Restore

```sh
# Brewfile
brew bundle

# npm global
xargs npm install --global < ./npm-global-list
```

## Backup Schedule

Everyday 06:00
